# Mabi_CN_to_Kr_Trans
중문-한글 번역 작업

참여 방법은 손번역, 번역기 모두 가능합니다.

참여 방법은 번역을 원하는 파일에 접근 뒤 오른쪽 위에 연필 모양을 누르면 수정 가능합니다.
